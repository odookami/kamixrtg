This module helps the user clear unwanted data using wizard with single click.

Data will be deleted only if the table exist in the database. For example, If Sale module is not installed, Sale checkbox will be visible on the popup but as the table doesn't exist it will have no effect.

Notes: Do not use this module in live environment.
Data is deleted via POSTGRESQL query and you will not be able to restore it.
This module supports only default odoo objects/class of the odoo not custom objects/class.

Key Features

Easily clear following table's data using the checkboxes:
1.User can clear the data with one checkbox selection of all data.
2.Customers & Vendors
3.Transfers
4.Sales & Deliveries
5.Purchase & Shipments
6.Journal Entries
7.Invoices & Payments & Journal Entries
8.Chart Of Accounts & All Accounting Data

To view the clear data feature user needs to enable this group.

Once user will have this group enabled then only the menu will be visible for the clear data.

After clicking the menu, user will get this option of objects, user need to select the option which records they needs to delete.
